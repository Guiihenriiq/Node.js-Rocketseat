# Curso de Node.js - Rocketseat

Este repositório contém os códigos e exercícios desenvolvidos durante o curso de Node.js oferecido pela Rocketseat. O curso cobre os fundamentos do Node.js, bem como conceitos e práticas essenciais para o desenvolvimento de aplicações backend com JavaScript.

### Explicação do conteúdo:

1. **Introdução:** Repositório com a intenção de organizar melhor o desenvolvimento abrangido durante o treinamento
2. **Estrutura do repositório:** Será dividida por modulos...
3. **Como Rodar o Projeto**
- [Node.js](https://nodejs.org) (versão recomendada: LTS)
- [NPM](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/) (gerenciador de pacotes)
